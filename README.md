# จุดเริ่มต้น
ก่อนที่ผมจะรู้จักกับ สอวน ผมก็เป็นแค่เด็กธรรมดาคนนึงในรร อินเตอร์ขนาดกลาง เก่งคณิตนิดหน่อยในรร แต่ออกมาก็โดนตบยับอยู่ดี 555 ทีนี้ตอนปิดเทอมปีใหม่ม 1 ว่างจัด แม่ผมเลยให้พี่ที่อยู่จะฬาคนนึงมาสอน python แต่พี่เค้าเป็นเด็กสอวน เลยรู้จักสอวน คอมจากพี่เค้าแหละครับ (เค้าชวน) แล้วผมก็เลยตั้งเป้าว่าจะเข้าค่ายให้ได้ทั้งๆที่ยังไม่รู้เลยว่าเข้าค่ายเป็นยังไง ได้อะไร 555 ผมก็เรียนคณิตมาเรื่อยๆเพท่อใช้สอบเข้า ช่วงแรกๆก็ดูเองผ่านเน็ตเอา ก็เรียนเนื้อหาผ่าน YT ทําโจทย์ผ่าน monkeyeveryday repeat ไปจบเก็บบทหลักๆครบ (ยกเว้นเรขาาาาาาาาาาาาาา ทําให้ผมกากเรขาถึงทุกวันนี้)

# ค่าย 1
ก็ผมเข้ามาโดยไม่รู้เลยว่ามันจะเป็นยังไง เข้ามากับเพื่อนิีกคนที่เคยอยู่รร เดียวกันเลยไม่ได้เหงาขนาดนั้น อยู่ดีกินดีมากกกก แบบอาหารค่อนข้าง very good  ก็เรียนไปครับ ครูที่นี่ friendly เป็นกันเองสุดๆแบบสอนนักเรียนไม่ฟัง 555 พตตก็ไม่ได้อะไรจากค่ายขนาดนั้นเพราะเรียนทุกอย่างมาแล้วๆเค้าก็สอบค่อนข้าง basic, ได้เพื่อนใหม่คนนึงที่เป็นแบบมหาเทพคณิตได้เรียญเงิน TMO ตั้งแต่ม 2 แบบโคตรอัจฉริยะอะ any ways ถึงท้ายค่ายเค้าพาไปเลี้ยงบุฟเฟต์ very good แบบโคตร premium
